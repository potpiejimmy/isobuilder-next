Getting Started

  1. Paste your ISO 8583 message (in hexadecimal format) into the main text field
  2. The application will automatically parse the message and display:
    • Message Type Indicator (MTI)
    • Bitmap visualization (both primary and secondary if present)
    • Individual data elements with their field numbers and labels
  3. Edit any field by clicking on it and modifying the value
  4. Changes are reflected in real-time in the hex representation

üí° Tip: When pasting into the hexadecimal representation, the application automatically detects whether your message is plain hexadecimal or ASCII text with <0x...> hexadecimal blocks and converts the ASCII parts to EBCDIC as needed. It also recognizes EMV-type transactions and adjusts field formats accordingly.

Supported Fields

The application supports all standard ISO 8583 fields including:

  • Transaction fields: Amount, date/time, trace numbers
  • Card data: PAN, track data, expiry date
  • Terminal data: Merchant type, terminal ID, location
  • Security: PIN data, MAC, CVV
  • EMV data: ICC chip data (field 55 with TAG-LENGTH-VALUE parsing)
  • Acquirer/Issuer data: Response codes, reference numbers, settlement data
